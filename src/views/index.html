Sobre o Projeto

O Jogo Real de Ur é um dos jogos de tabuleiro mais antigos já descobertos, originário da antiga Mesopotâmia.

Regras Básicas

O objetivo é percorrer o tabuleiro com suas 7 peças antes do adversário. Utilize os dados tetraédricos para mover.
