Sobre o Projeto

Esta página apresenta o desenvolvimento de uma versão web do antigo jogo de tabuleiro "The Royal Game of Ur".

O Jogo Real de Ur é um dos jogos de tabuleiro mais antigos já descobertos, originário da antiga Mesopotâmia.

Objetivos

O objetivo principal é criar uma aplicação web funcional que permita a dois jogadores (ou um jogador contra um bot) disputarem uma partida de Ur, seguindo suas regras.

Disciplina

Este projeto está sendo desenvolvido para a disciplina de INE5646 - Programação para Web.
